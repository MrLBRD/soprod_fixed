SoProd modifié

v1.6.2

Amélioration afin d'optimiser l'utilisation de SoProd. Dédié à l'utilisation du logiciel par l'équipe Maintenance Réactivité Premium/Essentiel

  • Correction de l'affichage des mots-clés

    Le bloc des mots-clés s'affiche sur toute la largeur du container afin d'avoir une meilleure visibilité du mot-clé.

  • Modèles de commentaire

    Ajout de divers boutons dans la zone de commentaire qui charge des modèles de commentaire : clôture de fiche, injoignable et appel classique.

  • CTRL+ENTRER = Poster le commentaire

    En étant dans le champ de commentaire, avec la combinaison de touche contrôle et entré, le commentaire sera publié de la même manière qu'au clic sur le bouton publier.

  • Sauvegarde du commentaire en cours

    Le commentaire en cours de rédaction est automatiquement sauvegardé dans une mémoire. Il est possible de le récupérer même après le rechargement de la page.

  • Affichage d'une horloge Dom-Tom

    Si le client est localisé dans un Dom-Tom alors une horloge est affiché avec l'heure chez le client.

  • Modification des tags d'événements

    Les étiquettes d'évenements dans les en-cours on étaient partiellement accordé avec ces mêmes évenements sur les agendas et les qualificatoins de fiche.

  • Clôture semi-automatique

    Si une fiche peut être clôturé ("CHECK MODIF TRAITEE") alors elle est automatiquement validé. Vous pouvez ensuite valider ou non la clôture, et le dernier message est affiché dans la modal de validation.

  • Bouton Portail ROI

    Sur les fiches premium, un bouton ouvrant le portail ROI est automatiquement ajouté avec les autres boutons.

  • Clic molette tableau de requêtes

    Pour ouvrir une fiche sur un nouvel onglet depuis le dashboard, il est possible de réaliser un clic avec la molette.

  • Menu contextuel

    Au clic droit sur un élément du dashboard, il est possible d'accèder a des fonctionnalités supplémentaire comme la copy facile pour Excel.
